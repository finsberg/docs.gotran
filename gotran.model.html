gotran.model.cellmodel module¶

class gotran.model.cellmodel.CellModel(name_, ns=None)[source]¶

Bases: gotran.model.ode.ODE

Basic class for storing information of a cell model

You can either initialize an empty CellModel similar to an ODE

(Maybe if we can create a library of th cellmodels then it the string matched that of the library we return the model in the library, e.g cell = CellModel(“Noble_1962”)` ?? ) Or you can initialize the cell using an existing ODE If you have an ODE and want to have a cell, then save the ODE to an *.ode file and load it using load_cell` (in stead of load_ode`)
